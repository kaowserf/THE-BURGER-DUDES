---

## 🌐 Website Prompt

> Build a bold, visually striking restaurant website for **The BurgerDudes** — a premium burger joint based in Fort Worth, Texas. The site should feel high-energy and mouth-watering, like a late-night food truck meets a classic American diner. Use a dark/charcoal background with punchy red and gold accents. Typography should be bold and unapologetic — think chunky display fonts for headings. Include smooth scroll, animated section reveals, hover effects on menu cards, and a sticky navigation bar. The full interactive menu should be browsable by category (tabs or sidebar). Include a combo deals banner, an order now CTA button, and a footer with hours, delivery info, and social links. Every menu item should display its name, description, price, and any badges (Bestseller, Spicy, New, Signature, etc.).

## 🛠️ Tech Stack Recommendations

- **Framework:** React or plain HTML/CSS/JS
- **Fonts:** Bold display font (e.g., Bebas Neue, Black Han Sans) + clean body font
- **Animations:** CSS transitions + scroll reveal (Intersection Observer)
- **Menu UX:** Tab-based category navigation with sticky header
- **Mobile:** Fully responsive, thumb-friendly CTAs
- **Theme:** Dark base (#1A1A1A) with red (#C0392B) and gold (#F0A500) accents
